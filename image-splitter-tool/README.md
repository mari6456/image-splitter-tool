# 画像かんたん切り抜き & 背景透明ツール

画像に並んだイラスト素材を**自動で1枚ずつに切り分け**、白などの背景を**ワンタップで透明化**して、PNG / WEBP でまとめて保存できる、**ブラウザだけで完結**する無料ツールです。インストール不要・登録不要・画像は外部に送信されません。

> 🔗 **デモ**: https://mari6456.github.io/image-splitter-tool/ （GitHub Pages 有効化後に使えます）

## 主な機能

- **自動切り分け** — 画像の縁から背景色を自動推定し、並んだイラストを1つずつ検出して切り抜き枠を作成
- **分割の細かさ** — 「こまかく / 標準 / まとめる」で検出粒度を調整
- **手動調整** — 枠のドラッグ移動・四隅ハンドルでのリサイズ・手描き追加・均等グリッド分割
- **背景透明化** — 消す色の指定（プリセット / スポイト / カラーピッカー）、消す強さ、フチのなめらかさ（フェザー）
- **保存** — 等倍 / 2倍 / 3倍、PNG（高画質）/ WEBP（軽量・品質調整可）、1枚ずつ保存・クリップボードコピー・ZIP一括保存
- **やさしいUI** — 日本語のわかりやすい表記、ライト / ダークテーマ対応、スマホ対応
- **どこでも保存** — 共有リンクやスマホ（iOS）でも保存できるフォールバックを内蔵

## 使い方

1. 画像をドラッグ＆ドロップ（またはクリックで選択 / `Ctrl+V` で貼り付け）
2. 自動で切り抜き枠ができます。必要なら枠を微調整
3. 右パネルで背景の透明化・保存サイズ・形式を設定
4. 「まとめて保存（ZIP）」または各画像の「保存」でダウンロード

はじめての方は、右パネルの「おためし画像を読み込む」ですぐに試せます。

## 技術メモ

- 依存ライブラリなしの**単一HTMLファイル**（`index.html`）。ビルド不要
- すべての処理は**クライアントサイド（ブラウザ内）**で完結。画像はサーバーへ送信されません
- 自動検出：ダウンサンプル上での連結成分ラベリング＋境界からの背景色推定＋実ピクセル基準のタイトトリミング
- ZIP：無圧縮（store）方式を内製実装（外部依存なし）
